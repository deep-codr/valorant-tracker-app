# 🎮 Valorant Match Tracker App

This is a clean and modern **Streamlit web app** that helps Valorant players track their match performance using a Riot ID.

> ⚠️ This app currently works with pre-downloaded sample JSON data placed in `data/riot_sample_data/`.

---

## 📌 Features

- Riot ID input to fetch match data
- Displays Total Matches, Average Kills/Deaths, Win Rate
- Beautiful line chart of K/D/A performance
- Full match breakdown in a clean table
- Stylish UI with dark theme and Valorant colors
- Fully modular and ready for real API integration
